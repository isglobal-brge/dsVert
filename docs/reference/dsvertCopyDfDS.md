# Copy a data frame to a new name (test helper)

Copy a data frame to a new name (test helper)

## Usage

``` r
dsvertCopyDfDS(data_name, output_name)
```

## Arguments

- data_name:

  Character. Name of the data frame symbol on the server.

- output_name:

  Character. Name to bind the output object to in the server-side
  environment.
