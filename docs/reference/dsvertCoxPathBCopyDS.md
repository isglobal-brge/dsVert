# Copy a session slot to another slot (alias helper for the Beaver plumbing)

Copy a session slot to another slot (alias helper for the Beaver
plumbing)

## Usage

``` r
dsvertCoxPathBCopyDS(source_key, target_key, session_id = NULL)
```

## Arguments

- source_key, target_key:

  session slot names.

- session_id:

  MPC session id.

## Value

list(copied = TRUE).
