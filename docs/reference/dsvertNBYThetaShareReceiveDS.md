# NL-side: receive (y + theta) share blob + store under canonical key

Decrypts the transport blob from `dsvertNBYThetaShareDS` and stores in
`ss$k2_nb_yt_share_fp`.

## Usage

``` r
dsvertNBYThetaShareReceiveDS(peer_yt_share_blob_key, session_id = NULL)
```

## Arguments

- peer_yt_share_blob_key:

  Character.

- session_id:

  Character.
